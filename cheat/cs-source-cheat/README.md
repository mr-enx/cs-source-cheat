# GameTrainer (Menu Mod)

این پروژه یک **مود منو** است که صرفاً جهت **آموزشی و تمرینی** ساخته شده است.

استفاده از این پروژه فقط در **نسخه‌های غیر اورجینال** امکان‌پذیر است.  
در نسخه‌های واقعی/آنلاین امکان **بن شدن** وجود دارد.

## نصب پیش‌نیازها
```bash
pip install pyinstaller
خروجی گرفتن (Export) به صورت فایل اجرایی (EXE)
python -m PyInstaller --onefile --windowed --uac-admin --icon=icon.ico --name "GameTrainer" main.py
